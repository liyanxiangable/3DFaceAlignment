# 3DFaceAlignment
三维面部三角网格模型的特征点定位



将三维面部模型降维平面并定位特征点，之后映射到三维，以提取面部信息。

### 使用方法

``` shell
python FaceAlignment.py -o ./resources/objModel.obj -t ./resources/texture.png
```
